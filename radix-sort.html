Best Case
O(n log n)
Pivot selalu membagi array menjadi dua bagian seimbang. Kedalaman rekursi = log₂ n.
Average Case
O(n log n)
Rata-rata pivot membagi cukup seimbang. Performa ini terjadi di sebagian besar kasus nyata.
Worst Case
O(n²)
Pivot selalu elemen terkecil/terbesar (array sudah terurut). Rekursi sedalam n level.
